À propos de YT Downloader

Un projet open source construit avec passion, pensé pour rester simple, rapide et respectueux de vos données.

Notre mission

Rendre le téléchargement vidéo accessible à tous. Aucune publicité, aucune donnée collectée, aucune inscription requise. YT Downloader est un outil pur, conçu pour fonctionner sans compromis.

Open source

Sous licence MIT, le code source est entièrement public. Chaque ligne est auditable, chaque contribution est la bienvenue. La transparence est au coeur du projet.

Stack technique

Tauri 2

Framework bureau léger et performant. Binaire minimal, interface native, sécurité par défaut.

Rust

Backend solide et sans garbage collector. Gestion mémoire garantie à la compilation, performances maximales.

yt-dlp

Moteur de téléchargement mature et maintenu. Supporte des centaines de sites, formats et qualités.

HTML / CSS / JS

Interface web classique. Pas de framework JS lourd, réactivité vanilla, chargement instantané.

GitHub Actions

CI/CD automatisé. Build multi-plateforme, signalement automatique des releases, tests continus.

Tailwind CSS

Design utility-first. Composants cohérents, dark mode natif, responsive sans effort.
